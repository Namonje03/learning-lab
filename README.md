# React Hooks Practice Project

This is a simple React project to practice basic hooks like:

- useState
- useEffect
- useRef


#  Features

## 1. Counter (useState)
A simple counter that increases, decreases, and resets a number.

## 2. Page Load Message (useEffect)
Displays a message in the console when the page loads.

## 3. Input Focus (useRef)
Automatically focuses an input field when a button is clicked.
